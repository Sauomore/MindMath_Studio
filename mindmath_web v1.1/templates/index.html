🧊 MindMath Studio

一个集数据清洗、方程建模、数值求解与可视化于一体的全功能科学计算平台。
支持代数方程、常微分方程（ODE）、偏微分方程（PDE）的交互式求解，并提供独立的线性回归工具。采用磨砂玻璃质感的 Web 界面，兼顾美观与实用。

🐍 Python 3.8+🌶️ Flask📊 Chart.js🧮 SciPy / NumPy📈 Scikit-learn

✨ 功能特性

📊 数据处理工坊

  • CSV/TSV 导入导出（自动识别分隔符，大文件线程化导入）
  • 可编辑数据表格（添加/删除行列、双击单元格编辑）
  • 缺失值处理：检测、均值/中位数/众数/常数/前向/后向/插值填充
  • 异常值处理：基于 IQR 或 Z-score 检测，一键清除
  • 行多选：任意选择数据行参与建模
  • 新增：数据筛选、描述性统计（均值、分位数、偏度、峰度）

📐 方程编辑工坊

  • 50+ 预设方程库（线性、多项式、指数、对数、逻辑斯蒂、周期函数、物理/化学/金融方程）
  • 自定义方程保存
  • 方程搜索
  • 参数映射（常数、数据列、x、t）
  • 实时预览曲线
  • 变量范围设置
  • 新增：非线性曲线拟合（自定义方程，输出参数、R²、RMSE、AIC、BIC）

📈 结果输出工坊

  • 代数方程 / ODE / PDE 求解
  • 多种图表：折线图、散点图、条形图、阶梯图、填充图、残差图、Q-Q图
  • 拟合统计量：R²、RMSE、AIC、BIC
  • PDE 专用热力图 + 时间切片
  • 结果导出 CSV

📉 独立线性回归工具

  • 列模式（X列+Y列）
  • 单列模式（行序号为X）
  • 行模式（列序号为X）
  • 完整统计量 + 散点拟合图

🤖 机器学习集成

  • 分类（逻辑回归、决策树、随机森林、SVM、KNN）
  • 回归（线性回归、岭回归、Lasso、决策树、随机森林、SVR、KNN）
  • 聚类（K-Means）
  • 模型持久化、预测功能

🛠 技术栈

  • 后端：Python 3.8+, Flask, Flask-CORS
  • 科学计算：NumPy, Pandas, SciPy, Scikit-learn
  • 前端：HTML5/CSS3 (Grid/Flex, backdrop-filter), JavaScript (ES6), Chart.js
  • 打包：PyInstaller (可选)

  • X自变量，Y因变量，t时间
  • 符号：· (乘), ÷ (除), ^ (幂), √ (根号)

  • ODE & PDE: dy/dt, ∂u/∂t, ∂²u/∂x²

常见问题

  • 如果出现报错“错误: '__import__'”怎么办？
    请检查变量的取值范围，当x小于等于0时可能会出现该错误。请适当调整X范围或参数。
  • 变量范围在哪里设置？
    在“方程编辑”页面的右侧，有X范围和T范围的输入框，可以手动调整。
  • 如何手动添加数据？
    在“数据工坊”页面，点击“添加列”可增加新列，点击“添加行”增加新行，双击单元格即可编辑内容。
  • 线性回归如何使用？
    在“方程工坊”页面底部有独立线性回归工具。列模式需要选择X和Y列；单列模式只需选择Y列（X自动为行序号）；行模式选择一行数值进行分析。
  • 非线性拟合如何使用？
    选择X列、Y列，输入方程表达式（如 a * exp(b * x) + c），点击开始拟合，结果会在结果页面显示。
  • 如何保存项目？
    点击左侧导航栏的“保存项目”，会将当前数据、方程、参数映射保存为JSON文件；点击“加载项目”可恢复。
  • 机器学习如何训练模型？
    选择模型类型（分类/回归/聚类）、算法、目标列和特征列，点击训练模型，结果会显示评估指标。
  • 如何使用已训练的模型进行预测？
    在机器学习页面下方选择已训练模型，点击“预测（使用当前数据）”或上传CSV文件进行预测。
  • PDE热力图不显示怎么办？
    请确保选择了正确的PDE方程（如热传导、波动方程），并设置合理的空间和时间范围。如果仍不显示，请检查浏览器控制台错误。
  • 为什么求解失败？
    可能原因：方程表达式有误、参数未正确映射、数据点不足。请检查方程语法和参数设置。
  • 软件是否可以打包成exe？
    可以。使用PyInstaller将Flask应用和前端打包，或使用Electron包装。详细方法请参考文档。
