# 更新日志

## v1.1 - 机器学习集成与界面全面优化

### ✨ 新增功能
- **机器学习工坊**：
  - 支持分类（逻辑回归、决策树、随机森林、SVM、KNN）
  - 支持回归（线性回归、岭回归、Lasso、决策树回归、随机森林回归、SVR、KNN）
  - 支持聚类（K-Means）
  - 模型训练后输出评估指标（准确率、混淆矩阵、分类报告、R²、RMSE、轮廓系数等）
  - 模型持久化与预测（支持使用当前数据或上传新 CSV 进行预测）
- **非线性曲线拟合**：用户可输入自定义方程（如 `a * exp(b * x) + c`），使用 `scipy.optimize.curve_fit` 自动拟合，输出参数、R²、RMSE、AIC、BIC 及拟合曲线图。
- **数据筛选**：支持按条件（>、>=、<、<=、==、包含文本）筛选行。
- **描述性统计**：一键生成所有数值列的均值、标准差、分位数、偏度、峰度，以表格形式展示。
- **项目保存/加载**：将当前数据、方程、参数映射保存为 JSON 文件，支持下次加载恢复。

### 🎨 界面优化
- **磨砂玻璃质感**：全面应用半透明背景、模糊效果、柔和阴影，提升视觉体验。
- **响应式布局**：适配不同屏幕尺寸，手机/平板下自动切换单列布局。
- **数据表格**：
  - 修复表格白色间隙问题（`border-collapse` 与背景冲突）
  - 支持固定表头（可开关），滚动时表头保持可见
  - 限制表格最大高度（`max-height: 65vh`），避免无限延伸
  - 单元格长文本自动换行
- **图表**：所有 Canvas 图表（预览、结果、PDE 热力图）自适应缩放，不再溢出容器。
- **左侧导航栏**：底部增加“保存项目”、“加载项目”按钮，与主功能按钮分离。

### 🐛 修复
- 修复线性回归单列模式中缺失值处理导致的缩进错误。
- 修复 PDE 求解时热力图生成的索引错误。
- 修复非线性拟合时方程参数检测失败的问题。
- 修复保存项目时 `NaN` 导致 JSON 序列化失败的错误（自动转换为 `null`）。
- 修复描述性统计表格列数过多时内容混乱的问题（增加水平滚动条）。
- 修复固定表头在动态重新渲染后失效的问题。
