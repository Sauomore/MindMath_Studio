**版本**: v1.6 (Web 版)

## 📌 当前已实现功能

- ✅ 数据处理工坊（CSV/TSV 导入导出，可编辑表格，缺失/异常检测，均值/中位数/众数填充，行多选）
- ✅ 方程编辑工坊（50+ 预设方程库，自定义方程保存，参数映射（常数/数据列/x/t），实时曲线预览）
- ✅ 结果输出工坊（代数/ODE/PDE 求解，多种图表（折线、散点、条形、阶梯、填充、残差、Q-Q），拟合统计量 R²/RMSE/AIC/BIC）
- ✅ PDE 专用可视化（热力图 + 时间切片滑块）
- ✅ 独立线性回归工具（列模式、单列模式、行模式）
- ✅ 磨砂玻璃质感 UI + 极简无边框设计
- ✅ Flask 后端 + 纯前端 SPA 架构
- ✅ 项目介绍、关于、常见问题模态框

## 🚀 后续更新路线（按优先级排序）

### 第一阶段：数据处理增强（v1.7）

| 功能 | 描述 | 预计完成时间 |
|------|------|--------------|
| **数据筛选** | 支持按条件（>、<、=、包含等）筛选行 | 1 周 |
| **描述性统计** | 一键生成所有数值列的均值、标准差、分位数、偏度、峰度 | 1 周 |
| **列运算** | 通过表达式创建新列（例：`新列 = 列A * 2 + 列B`） | 1 周 |
| **数据排序** | 按单列或多列升序/降序排序 | 3 天 |

### 第二阶段：统计分析模块（v1.8）

| 功能 | 描述 |
|------|------|
| **相关性分析** | 计算数值列之间的 Pearson / Spearman 相关系数矩阵，绘制热力图 |
| **假设检验** | 单样本 t 检验、双样本 t 检验、卡方检验、方差分析（ANOVA） |
| **箱线图 / 小提琴图** | 用于展示数据分布的可视化组件 |

### 第三阶段：非线性拟合与方程扩展（v1.9）

| 功能 | 描述 |
|------|------|
| **自定义曲线拟合** | 支持用户输入任意表达式（如 `a * exp(b * x) + c`），使用 `scipy.optimize.curve_fit` 拟合，输出参数、R²、RMSE、AIC、BIC |
| **参数置信区间** | 显示拟合参数的 95% 置信区间 |
| **多方程对比** | 在同一图表中比较两个以上方程的拟合曲线，并列出 AIC/BIC 排序 |

### 第四阶段：机器学习集成（v2.0）

| 功能 | 描述 |
|------|------|
| **分类/回归模型** | 集成逻辑回归、决策树、随机森林、SVM（基于 scikit-learn），支持训练/测试集划分 |
| **聚类分析** | K-Means、层次聚类，可视化聚类结果（散点图） |
| **特征重要性** | 对树模型输出特征重要性条形图 |
| **模型评估指标** | 准确率、F1、ROC 曲线、混淆矩阵 |

### 第五阶段：工作流与自动化（v2.1）

| 功能 | 描述 |
|------|------|
| **项目保存/加载** | 将当前数据、方程、参数映射保存为一个 `.mindmath` 文件，下次一键恢复 |
| **脚本录制与回放** | 记录用户操作（导入 → 清洗 → 求解 → 导出），生成 Python 脚本，支持批量处理 |
| **批量处理** | 对文件夹内所有 CSV 文件执行同一建模流程，输出汇总报告 |

### 第六阶段：可视化增强与协作（v2.2+）

| 功能 | 描述 |
|------|------|
| **3D 表面图** | 使用 Plotly 实现可旋转的 PDE 3D 视图 |
| **动态交互** | 鼠标悬停显示数值，缩放/平移图表 |
| **PDF/HTML 报告生成** | 导出包含图表、统计量、模型参数的完整报告 |
| **API 服务** | 将求解器暴露为 REST API，供其他程序调用 |
